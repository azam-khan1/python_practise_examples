## Topics covered so far

- **Data types** — lists, tuples, sets, dictionaries (with real use cases, not just theory)
- **Control flow** — conditions, match-case, ternary expressions
- **Loops** — for, while, enumerate, zip, the break/else pattern
- **Functions** — parameters, return values, nested functions, scope
- **Recursion** — sum of n, reversing a list recursively
- **OOP** — writing classes, constructors, creating objects
- **File handling** — opening, reading, writing, appending files
